## To start the appliccation
    
    npm init

## To install nodemon
    
    npm i nodemon

## To run the application

    npm run dev


## HTTP methods
    GET         => To get relative data from the server
    POST        => Set the data to the server
    DELETE      => Deleting the data from the server
    PATCH       => Update certain feilds
    PUT         => Overwriting / Full update
